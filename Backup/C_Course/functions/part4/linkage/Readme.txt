The term used by the Standard to describe the accessibility of objects from one file to another, or even within the same 
file, is linkage.

Internal linkage
	The objects with this linkage can be refer only with in the file in which it is defined. They are not shared with other Files. They are Private to file

External linkage
        The objects with this linkage can be refer throwout the program . They can be shared with other Files. 
	They are Public to all files.
